# Item Search Function


Item Search Function는 다음과 같이 구성된다.
```
- Market Summary
- Search for item name 
	- ( 한/영 모두 지원 Ex: 삼성전자 , Samsung | 부분 검색 가능 )

- Search for item code 
	- ( 숫자만 지원 Ex: 005930 | 정확한 검색만 지원 )

- auto-completion search
    - ( 2글자 이상 입력 시 드롭다운 자동완성 표시 )
    - ( 최대 10개 결과 노출 )
    - ( 검색어 하이라이팅 )


- Show popular items
    - ( 실시간 검색 랭킹 TOP 10 표시 시장 조회수 기반 )
    - ( 코스피 / 코스닥 분리 탭 )

- Resent Serch 
    - ( 최근 검색 종목 최대 10개 로컬 저장 )
    - ( 개별 삭제 / 전체 삭제 )
    - ( 검색창 포커스 시 최근 기록 자동 표시 )

```
